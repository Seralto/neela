# Neela [WIP]

## New Elegant Language

Neela is an interpreted programming language.

## Language specifications

### Comments
```
// This is a comment
```

### Send to standard output (AKA print)
```
out "Hello World"
out 42
```

### Assignment
```
a = 42
str = "Neela is cool!"
```

### Conditionals
```
if percent == 100
  out "You got everything"
.
```

### Functions
```
fun sayHello()
  out "Hello!"
.
```
